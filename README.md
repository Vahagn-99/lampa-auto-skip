# ⏭ Skip Intro/Outro — плагин для Lampa

Автоматический пропуск заставок, рекапов, титров и превью в сериалах для медиаплеера [Lampa](https://lampa.app).

## Возможности

- **3 источника данных** — TheIntroDB, IntroDB, IntroHater. Плагин проходит по цепочке API и находит таймкоды даже для редких сериалов
- **Умное обнаружение** — если сериала нет ни в одной базе, плагин определяет заставку по анализу субтитров (поиск длинной паузы) и звуковой дорожке (Web Audio API)
- **Параллельная работа** — API-запросы и локальная детекция запускаются одновременно, результаты объединяются. Если API даёт неточный таймкод, а детекция находит заставку раньше — берётся лучший вариант
- **Smart Auto-Skip** — если вы один раз пропустили заставку, плагин запомнит это и в следующих сериях будет пропускать автоматически с обратным отсчётом (4 сек). Можно отменить кнопкой «Назад»
- **Поддержка TV-пультов** — интеграция с `Lampa.Keypad` для корректной работы на Samsung Tizen, LG WebOS, Android TV. Настраиваемые кнопки пропуска/отмены
- **Кеширование** — результаты API кешируются в localStorage, результаты детекции — в Lampa.Storage (для всего сезона)
- **4 типа сегментов** — заставка (intro), рекап (recap), титры (credits), превью (preview). Каждый можно включить/выключить отдельно

## Установка

### Способ 1: URL (рекомендуется)

1. Откройте Lampa → **Настройки** → **Дополнения** → **Добавить**
2. Вставьте URL:

```
https://vahagn-99.github.io/lampa-auto-skip/init.js
```

3. Нажмите **Добавить** и перезагрузите приложение

### Способ 2: Ручная установка

1. Скачайте файл `init.js` из репозитория
2. Разместите на своём сервере или добавьте локально

## Настройки

После установки в Lampa появится раздел **«Пропуск заставок»** в настройках:

| Параметр | Описание | По умолчанию |
|----------|----------|:------------:|
| Включить плагин | Показывать кнопку пропуска | ✅ |
| Всегда автопропуск | Перематывать без кнопки для всех сериалов | ❌ |
| Умное обнаружение | Определять заставку по субтитрам и звуку | ✅ |
| Пропускать заставку | Пропуск intro | ✅ |
| Пропускать рекап | Пропуск recap | ✅ |
| Пропускать титры | Пропуск credits | ✅ |
| Пропускать превью | Пропуск preview | ❌ |
| Кнопка «Пропустить» | Enter, Пробел или цветные кнопки пульта | Enter/OK |
| Кнопка «Отменить» | Назад или цветные кнопки пульта | Назад |

│              Запуск серии                        │

        ┌────────────┴────────────┐

  │ API цепь │            │  Детекция    │

  │    ↓     │            │ 2. Аудио     │
  │ IntroDB  │            │              │

       └────────────────┘

## API источники

| # | Сервис | Endpoint | Описание |
|---|--------|----------|----------|
| 1 | TheIntroDB | `api.theintrodb.org/v2` | Основной. Поддерживает intro, recap, credits, preview |
| 2 | IntroDB | `api.introdb.app` | Резервный. Intro и credits |
| 3 | IntroHater | `introhater.com/api` | Дополнительный. Комьюнити-база с верификацией |

## Разработка

Исходный код: `skip-intro.js` (полная версия с комментариями)
Продакшен: `init.js` (минифицированная версия)

```bash
# Минификация
npx terser skip-intro.js --compress --mangle --output init.js
```
